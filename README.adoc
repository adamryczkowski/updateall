A Python-based system for orchestrating updates across multiple package managers and system components.

toc::[]

== Overview

Update-All is a modular system that coordinates updates for:

* APT packages
* Pipx applications
* Flatpak applications
* And more (via plugins)

== Project Structure

This is a monorepo with independent subprojects:

[source]
----
update-all/
├── core/           # Core library (models, interfaces, utilities)
├── cli/            # Command-line interface
├── plugins/        # Plugin implementations (apt, pipx, flatpak, etc.)
├── ui/             # Rich terminal UI components
├── stats/          # Statistics and reporting
└── docs/           # Documentation
----

== Requirements

* Python 3.11+
* Poetry 2.0+
* just (command runner)
* pre-commit

== Quick Start

[source,bash]
----
# Install dependencies for all subprojects
just install-all

# Run validation
just validate

# Run the CLI
just run
----

== Development

Each subproject is an independent Poetry project with its own:

* `pyproject.toml` - Project configuration
* `poetry.toml` - Poetry settings (in-project venv)
* `justfile` - Common development commands

=== Common Commands
